# Capture (shyla-T4SG-summer-deliverable)

***Save the moment with Capture***
Capture is a platform where users can share film-style photos online with others. Users can take photos and edit them with our built-in photobooth. Capture’s photobooth removes the stress of trying to get the “perfect” shot to post online and instead allows users to save the moment with the simple snap of an "instant" camera.

## Tech Stack

- React
- Next.js
- TypeScript
- Supabase
- Supabase Auth
- At least one API call
- Git/GitHub
- Vercel deployment

## Features

- Integrated camera
- Photo editing + stickers
- Photo sharing
